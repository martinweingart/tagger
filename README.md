# Tagger

## Uso

tagger -f archivo <br>
tagger -r carpeta

## Parámetros

-t título --title <br>
-a artista --artist <br>
-b album --album <br>
-y año --year <br>
-g genero --genre <br>
-n número pista --track

## Dependencias

pip install mutagen
